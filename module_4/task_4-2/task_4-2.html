  2. 2. What does it mean to "hardcode" when we talk about programming? (In English, it's called "hard coding", "hard-coding", or "hardcoding").

    3. Hardcoding means explicitly writing values directly into the code rather than dynamically generating or retrieving them at runtime. For example, writing let numbers = [1, 2, 3]; is hardcoding, while generating the array dynamically based on user input or calculations is not. Hardcoding is simple but lacks flexibility.
